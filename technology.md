# 🛠️ Technologies Used

## 🌐 Frontend

### HTML5
Used for structuring the web application.

### CSS3
Used for styling and layout design.

### Bootstrap 5
- Responsive design
- Prebuilt UI components
- Grid system

### JavaScript (ES6)
- DOM manipulation
- Event handling
- Logic implementation

### Axios
- Used to make HTTP requests to backend API
- Handles GET, POST, PUT, DELETE operations

---

## ⚙️ Backend

### Node.js
- JavaScript runtime environment
- Handles server-side logic

### Express.js
- Web framework for Node.js
- Creates REST APIs
- Handles routing and middleware

---

## 🗄️ Data Storage

### JSON File (`employees.json`)
- Acts as a simple database
- Stores employee records
- Read/write using Node.js `fs` module

---

## 🔧 Tools

### Postman
- API testing
- Debugging backend endpoints

### VS Code
- Code editor
- Extensions for development

---

## 🔄 Communication

### REST API
Frontend communicates with backend using RESTful APIs.

---

## 📊 Concepts Used

- CRUD Operations
- REST API Design
- Asynchronous Programming
- Pagination
- Filtering & Searching
- MVC-like structure

---

## 🚀 Summary

This project demonstrates a **full-stack application** using:

- Frontend (UI + Axios)
- Backend (Node + Express)
- Data handling (JSON)

It is ideal for learning **real-world CRUD applications**.
