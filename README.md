# 🚀 Employee Management System

A full-stack Employee Management System built using **Node.js, Express, HTML, CSS, Bootstrap, and JavaScript (Axios)**.

This application allows users to manage employee records with features like add, update, delete, search, filter, and pagination — all without page reload.

---

## 📌 Features

- ➕ Add new employees
- 👀 View all employees in a dynamic table
- ✏️ Edit employee details
- 🗑️ Delete employees with confirmation
- 🔍 Search employees by name (case-insensitive)
- 🎯 Filter by department
- 📄 Pagination (5 records per page)
- 📊 Department distribution stats
- ⚡ Fast UI (no page reload using Axios)

---
